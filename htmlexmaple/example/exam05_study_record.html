HTML 학습 기록

오늘은 HTML 문서의 기본 구조와 자주 사용하는 태그를 학습했습니다.

오늘 배운 내용

HTML 문서는 <html>, <head>, <body> 태그를 기본 구조로 가진다.

화면에 표시되는 대부분의 내용은 <body> 태그 안에 작성한다.

학습 상태

처음 해보니 HTML이 신기하다고 생각했다.

예제를 작성해보니 HTML이 재미있다고 생각했다.

간단한 코드 메모

HTML 기본 구조

< !DOCTYPE html>
<html>
<head>
    문서 정보 작성
</head>
<body>
    화면에 표시할 내용 작성
</body>
</html>

기억할 표현

문단 태그는 <p>내용</p>형태로 작성한다.

공백을 여러 칸 표시하고 싶을 때는      와 같이 작성할 수 있다.
