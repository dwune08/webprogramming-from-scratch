11. jQuery 기초

jQuery는 JavaScript로 DOM 요소 선택, 이벤트 처리, 스타일 변경, 애니메이션, Ajax 등을 간단하게 작성할 수 있도록 도와주는 JavaScript 라이브러리이다.

11.1 jQuery란?

jQuery는 JavaScript 코드를 더 짧고 간단하게 작성할 수 있도록 만들어진 라이브러리이다. 예전 웹 프로젝트에서는 DOM 조작, 이벤트 처리, Ajax 요청 등을 처리할 때 많이 사용되었다.

최근에는 표준 JavaScript 기능이 많이 좋아지고 React 같은 UI 라이브러리나 Vue 같은 프레임워크 사용이 많아졌지만, 기존 웹 사이트나 관리자 페이지에서는 jQuery 코드가 아직 사용되는 경우가 많다. 따라서 jQuery를 새 프로젝트의 중심 기술로 사용하기보다는 기존 코드를 읽고 수정하기 위한 목적으로 학습하면 좋다.

11.2 jQuery 파일 연결 방법

jQuery를 사용하려면 HTML 문서에 jQuery 파일을 먼저 연결해야 한다. 직접 다운로드한 파일을 연결할 수도 있고, CDN 주소를 이용하여 연결할 수도 있다.

CDN 방식

<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

직접 다운로드한 파일 연결 방식

<script src="../js/jquery-3.7.1.min.js"></script>

jQuery 문법을 사용하려면 직접 작성한 JavaScript 파일보다 jQuery 파일을 먼저 연결해야 한다.

11.3 기본 문법

jQuery는 $() 함수를 사용하여 HTML 요소를 선택한다. 선택자 문법은 CSS 선택자와 거의 동일하다. 

$('#아이디선택자')
$('.클래스선택자')
$('태그선택자')
jQuery 메서드 설명
.text() 요소의 텍스트 내용을 가져오거나 변경한다.
.html() 요소의 HTML 내용을 가져오거나 변경한다.
.val() 입력 요소의 값을 가져오거나 변경한다.
.css() 요소의 CSS 스타일을 변경한다.
.addClass() 요소에 클래스를 추가한다.
.removeClass() 요소에서 클래스를 제거한다.
.toggleClass() 클래스가 있으면 제거하고, 없으면 추가한다.
.on() 이벤트를 등록한다.
.show() 숨겨진 요소를 보여준다.
.hide() 요소를 숨긴다.
.toggle() 요소가 보이면 숨기고, 숨겨져 있으면 보여준다.

11.5 Ajax 개념

Ajax는 페이지 전체를 새로고침하지 않고 서버와 데이터를 주고받는 기술이다. jQuery에서는 $.ajax() 메서드를 사용하여 서버에 요청을 보낼 수 있다. 

// $.ajax(요청 정보).done(응답 성공 처리).fail(응답 실패 처리);
// $.ajax({ }).done(function(data){ });

$.ajax({
    url: "요청 주소",
    method: "GET",          // GET 또는 POST
    data: {
        name: "홍길동",      // 서버로 전달할 데이터
        age: 20
    },
    dataType: "json"        // text, xml, json 중 하나
}).done(function(data) {
    // 통신이 성공했을 때 실행된다.
    // 서버에서 응답한 데이터는 data 매개변수로 전달된다.
    console.log(data);
}).fail(function(xhr, textStatus) {
    // 통신이 실패했을 때 실행된다.
    console.log("요청 실패");
    console.log(textStatus);
});

url은 요청할 서버 주소이고, method는 요청 방식이다. data에는 서버로 보낼 값을 작성하고, dataType에는 서버에서 받을 데이터 형식을 작성한다.
