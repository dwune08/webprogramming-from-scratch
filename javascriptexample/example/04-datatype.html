3.4 변수와 자료형

자바스크립트에서 변수는 데이터를 저장하는 공간이고, 자료형은 변수에 저장되는 값의 종류를 의미한다. 자바스크립트는 변수 선언 시 자료형을 고정하지 않고, 저장되는 값에 따라 자료형이 결정된다.

3.4.1 변수 선언

변수는 데이터를 저장하기 위해 사용하는 이름이 있는 공간이다. 자바스크립트에서는 var, let, const를 사용하여 변수를 선언할 수 있다. 현재는 주로 let과 const를 사용한다. 

let 변수명 = 값,
const 상수명 = 값;
var 변수명 = 값;

개념 정리

let은 값을 다시 대입할 수 있는 변수를 선언할 때 사용한다. 같은 이름으로 재선언할 수 없고, 블록 레벨 스코프를 가진다.

const는 값을 다시 대입하지 않을 변수를 선언할 때 사용한다. 같은 이름으로 재선언할 수 없고, 재할당할 수 없으며 선언과 동시에 초기화해야 한다.

참고

var는 예전부터 사용하던 변수 선언 방식이다. 같은 이름으로 재선언할 수 있고, 함수 레벨 스코프를 가진다. 이러한 특징 때문에 코드의 흐름을 예측하기 어려운 경우가 있어, 새로운 코드를 작성할 때는 가급적 let과 const를 사용하는 것이 좋다.

3.4.2 기본 자료형

기본 자료형은 하나의 값을 직접 표현하는 자료형이다. 대표적으로 문자열, 숫자, 논리값, undefined, null, bigint가 있다.

string

문자열을 나타내는 자료형이다. 작은따옴표, 큰따옴표, 백틱으로 작성할 수 있다.

number

정수와 실수를 모두 나타내는 자료형이다.

boolean

true 또는 false 값을 가지는 논리 자료형이다.

undefined / null

undefined는 값이 할당되지 않은 상태이고, null은 값이 없음을 의도적으로 표현한 값이다.

bigint

bigint는 Number.MAX_SAFE_INTEGER보다 큰 정수처럼, 일반 number 자료형으로 정확하게 표현하기 어려운 큰 정수를 다룰 때 사용한다. 숫자 뒤에 n을 붙여 작성한다.

3.4.3 참조 자료형

참조 자료형은 여러 값을 하나의 구조로 묶어 다룰 때 사용한다. 대표적으로 배열, 객체, 함수가 있다. 배열과 객체의 typeof 결과는 object이고, 함수의 typeof 결과는 function이다. 

const 배열변수 = [값1, 값2, 값3];

const 객체변수 = {
    속성명1: 값1,
    속성명2: 값2
};

3.4.5 형 변환

형 변환은 값의 자료형을 다른 자료형으로 바꾸는 것을 의미한다. 사용자 입력값은 문자열로 처리되는 경우가 많으므로, 계산이 필요한 경우 숫자로 변환해야 한다. 자바스크립트에서는 Number(), String(), Boolean() 등을 사용하여 자료형을 변환할 수 있다. 

Number(문자열값)
String(숫자값);
Boolean(값);
