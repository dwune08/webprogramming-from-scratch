3.3 DOM 요소 접근 방법

자바스크립트는 HTML 요소를 선택한 뒤, 선택한 요소의 내용, 속성, 스타일을 변경할 수 있다.

3.3.1 DOM 개념

DOM(Document Object Model)은 HTML 문서를 자바스크립트에서 다룰 수 있도록 객체 형태로 표현한 구조이다. 브라우저는 HTML 문서를 읽고, 각 태그를 요소 객체로 만들어 계층 구조로 관리한다.

자바스크립트는 DOM을 이용하여 HTML 요소에 접근하고, 요소의 내용, 속성, 스타일 등을 변경할 수 있다.

개념 정리

자바스크립트에서 HTML 요소를 가져온다는 것은 해당 요소를 나타내는 객체를 가져온다는 의미이다. 가져온 요소 객체에 textContent, innerHTML, style, value 등을 사용하여 화면의 내용을 변경할 수 있다.

3.3.2 HTML 요소 접근 메서드

DOM에서 요소를 가져올 때 사용하는 대표적인 메서드는 다음과 같다.

document.getElementById('id속성값');
document.getElementsByName('name속성값');
document.getElementsByClassName('class속성값');
document.getElementsByTagName('태그명');

document.querySelector('CSS선택자');
document.querySelectorAll('CSS선택자');

3.3.3 id로 접근하기

getElementById()는 id 속성값을 이용하여 HTML 요소 하나를 가져오는 메서드이다. id는 요소를 구분하기 위한 이름이므로, 한 문서 안에서 같은 id를 여러 번 사용하지 않는다. 

const 변수명 = document.getElementById('id속성값');
변수명.style.CSS속성명 = '변경할값';

3.3.4 class로 접근하기

getElementsByClassName()은 같은 class 값을 가진 HTML 요소들을 가져오는 메서드이다. 여러 요소가 반환되므로 배열처럼 인덱스를 사용하여 각 요소에 접근할 수 있다. 

const 변수명 = document.getElementsByClassName('class속성값');
변수명[인덱스].style.CSS속성명 = '변경할값';

3.3.6 name으로 접근하기

getElementsByName()은 같은 name 값을 가진 HTML 요소들을 가져오는 메서드이다. 체크박스나 라디오 버튼처럼 관련된 입력 요소를 묶어 처리할 때 사용할 수 있으며, 여러 요소가 반환되므로 배열처럼 인덱스를 사용하여 각 요소에 접근할 수 있다.

3.3.7 CSS 선택자로 접근하기

querySelector()와 querySelectorAll()은 CSS 선택자를 사용하여 HTML 요소를 가져오는 메서드이다. querySelector()는 조건에 맞는 첫 번째 요소 하나를 가져오고, querySelectorAll()은 조건에 맞는 모든 요소를 가져온다. 

document.querySelector('CSS선택자');
document.querySelectorAll('CSS선택자');
