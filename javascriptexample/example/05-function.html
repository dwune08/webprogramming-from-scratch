3.5 함수

함수는 특정 기능을 수행하는 코드를 하나의 이름으로 묶어 놓은 것이다. 반복해서 사용하는 코드를 함수로 만들면 코드의 중복을 줄이고 유지보수를 쉽게 할 수 있다.

3.5.1 함수 선언문

함수 선언문은 선언적 함수라고도 하며, function 키워드 뒤에 함수 이름을 작성하여 함수를 정의하는 방식이다. 필요한 경우 매개변수를 전달받고, return을 사용하여 결과값을 반환할 수 있다. 

function 함수명(매개변수) {
    실행문;
    return 반환값;
}

3.5.3 익명 함수와 함수 표현식

익명 함수는 이름이 없는 함수이다. 익명 함수는 단독으로 직접 호출하기 어렵기 때문에 보통 변수에 저장하여 사용한다. 이렇게 함수를 값처럼 변수에 저장하는 방식을 함수 표현식이라고 한다. 

const 변수명 = function(매개변수) {

3.5.4 즉시 실행 함수

즉시 실행 함수는 함수를 정의하자마자 바로 실행하는 함수이다. 한 번만 실행할 초기화 코드나 변수 이름 충돌을 줄이고 싶을 때 사용할 수 있다. 

(function() {
    실행문;
})();

3.5.5 화살표 함수

화살표 함수는 ES6에서 추가된 간결한 함수 표현식이다. function 키워드 대신 => 기호를 사용한다. 

1. 익명 함수
const 함수명 = function() {
    실행명;
};

const 함수명 = function(매개변수) {
    return 반환값;
};

2. 화살표 함수
const 함수명 = () => {
    실행문;
};

const 함수명 = (매개변수) => {
    return 반환값;
};

3. 화살표 함수 축약형
- 매개변수가 하나이면 () 생략 가능
const 함수명 = 매개변수 => {
    실행문;
};

3.5.7 콜백 함수

콜백 함수는 다른 함수의 매개변수로 전달되는 함수이다. 매개변수로 전달된 함수는 즉시 실행될 수도 있고, 특정 작업이 끝난 뒤 필요한 시점에 호출될 수도 있다.

자바스크립트에서는 함수를 값처럼 변수에 저장하거나, 다른 함수의 인자로 전달할 수 있다.
이처럼 다른 함수에 전달되어 나중에 호출되는 함수를 콜백 함수라고 한다.
