3.8 자바스크립트 내장 객체

내장 객체는 자바스크립트에서 기본적으로 제공하는 객체이다. 문자열 처리, 배열 관리, 날짜와 시간 처리, 수학 계산, 브라우저 제어 등에 사용할 수 있다.

3.8.1 내장 객체의 개념

자바스크립트는 자주 사용하는 기능을 객체 형태로 미리 제공한다. 개발자는 이러한 내장 객체의 속성과 메서드를 사용하여 데이터를 쉽게 처리할 수 있다. 

객체명.속성;
객체명.메서드();

자주 사용하는 String, Date, Math, Array 객체를 중심으로 학습한다.

3.8.2 String 객체

String 객체는 문자열 데이터를 다룰 때 사용한다. 문자열의 길이를 확인하거나, 특정 문자를 찾고, 대소문자를 변경하고, 문자열을 나누는 기능을 제공한다. 

const text = "JavaScript";

text.length;
text.indexOf("찾을문자열");
text.substring(시작인덱스, 종료인덱스);
text.toLowerCase();
text.toUpperCase();
text.trim();
text.split("구분자");

3.8.3 Date 객체

Date 객체는 날짜와 시간 정보를 다룰 때 사용한다. 현재 날짜를 가져오거나, 선택한 날짜의 연도, 월, 일, 요일 정보를 확인할 수 있다. 

const today = new Date();

- 문자열로 날짜 생성
const selectedDate = new Date("2026-12-31");

- 숫자로 날짜 생성
: 월은 0부터 시작하므로 11은 12월을 의미한다.
const selectedDate = new Date(2026, 11, 31); 

selectedDate.getFullYear();
selectedDate.getMonth() + 1;
selectedDate.getDate();
selectedDate.getDay();
selectedDate.getTime();

3.8.4 Math 객체

Math 객체는 수학 계산에 필요한 속성과 메서드를 제공한다. 다른 객체와 달리 new로 생성하지 않고 Math 이름으로 바로 사용한다. 

Math.random();
Math.floor(숫자);
Math.ceil(숫자);
Math.round(숫자);
Math.max(값1, 값2, 값3);
Math.min(값1, 값2, 값3);

3.8.5 Array 객체

Array 객체는 여러 개의 값을 순서대로 저장할 때 사용한다. 배열은 인덱스로 값에 접근하며, push(), forEach(), join()과 같은 메서드를 자주 사용한다. 

const arrayName = [value1, value2, value3];

arrayName.length;
arrayName.push(newValue);

arrayName.forEach(function(value) {
    실행문;
});

arrayName.join("separator");
