3.10 DOM 요소 생성과 삭제

이 페이지에서는 JavaScript로 HTML 요소를 동적으로 생성하고, 화면에 추가하거나 삭제하는 방법을 학습한다.

3.10.1 DOM 노드 생성과 추가

DOM은 HTML 문서의 구조를 객체 형태로 표현한 것이다. 브라우저는 HTML 문서를 해석하여 DOM 트리를 만들고, 이 트리를 구성하는 각각의 요소를 노드(Node)라고 한다. 예를 들어 HTML 태그는 요소 노드, 태그 안의 글자는 텍스트 노드, 태그에 작성된 속성은 속성 노드로 표현된다.

JavaScript는 DOM 메서드와 속성을 사용하여 새로운 요소 노드와 텍스트 노드를 만들고, 원하는 부모 요소 안에 추가하거나 기존 노드를 삭제할 수 있다. 또한 선택한 요소를 기준으로 부모 요소, 자식 요소, 다음 형제 요소를 찾을 수도 있다.

document.createElement("태그명") 새로운 HTML 요소 노드를 생성한다.
document.createTextNode("문자열") 텍스트 노드를 생성한다.
부모요소.appendChild(자식요소) 부모 요소의 마지막 자식으로 요소를 추가한다.
부모요소.removeChild(자식요소) 부모 요소에서 지정한 자식 요소를 삭제한다.
요소.parentElement 선택한 요소의 부모 요소를 가져온다.
요소.children 선택한 요소의 자식 요소들을 가져온다.
요소.firstElementChild 선택한 요소의 첫 번째 자식 요소를 가져온다.
요소.lastElementChild 선택한 요소의 마지막 자식 요소를 가져온다.
요소.previousElementSibling 선택한 요소의 이전 형제 요소를 가져온다.
요소.nextElementSibling 선택한 요소의 다음 형제 요소를 가져온다. 
const 부모요소 변수 = document.getElementById("부모요소id");

const 요소변수 = document.createElement("요소명");
const 텍스트변수 = document.createTextNode("텍스트 내용");

요소변수.appendChild(텍스트변수);
부모요소변수.appendChild(요소변수);
