3.2 자바스크립트 데이터 입출력

자바스크립트에서는 알림창, 입력창, 콘솔, HTML 요소 등을 이용하여 데이터를 출력하거나 사용자로부터 값을 입력받을 수 있다.

3.2.1 window 객체와 document 객체

자바스크립트는 브라우저에서 실행될 때 여러 객체를 사용할 수 있다. 그중 window 객체는 브라우저 창 전체를 나타내는 최상위 객체이고, document 객체는 현재 브라우저에 표시된 HTML 문서를 나타내는 객체이다.

화면에 알림창을 띄우거나 입력창을 보여줄 때는 window 객체의 기능을 사용하고, HTML 요소를 찾거나 내용을 변경할 때는 document 객체의 기능을 사용한다.

개념 정리

alert(), confirm(), prompt()는 window 객체가 제공하는 메서드이다. 원래는 window.alert()처럼 작성할 수 있지만, window.는 대부분 생략하여 작성한다.

alert('출력할 메시지');
console.log('출력할 데이터');
document.write('문서에 출력할 내용');

선택한요소.textContext = '변경할 텍스트 내용';
선택한요소.innerHTML = 'HTML 태그를 포함한 내용';

3.2.5 템플릿 문자열

템플릿 문자열은 문자열 안에 변수나 계산 결과를 쉽게 넣을 수 있는 문자열 작성 방법이다. 기존에는 문자열과 변수를 연결할 때 + 연산자를 사용했지만, 템플릿 문자열을 사용하면 더 간단하고 읽기 쉽게 작성할 수 있다.

템플릿 문자열은 작은따옴표나 큰따옴표가 아니라 백틱(`)을 사용하여 작성한다. 문자열 안에 변수나 계산식을 넣을 때는 ${표현식} 형태로 작성한다. 

`문자열 ${표현식}`
