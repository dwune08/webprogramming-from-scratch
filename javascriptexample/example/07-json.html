3.7 JSON 데이터 활용

JSON은 데이터를 주고받거나 저장할 때 많이 사용하는 텍스트 기반의 데이터 형식이다. 실제 개발에서는 서버나 파일에서 전달받은 JSON 문자열을 자바스크립트 객체로 변환하여 사용한다. 이번 단원에서는 먼저 객체 형태의 데이터를 변수에 저장하고, 각 데이터에 접근하는 방법을 학습한다.

3.7.1 JSON 개념

JSON(JavaScript Object Notation)은 데이터를 key-value 쌍으로 표현하는 형식이다. JavaScript 객체 문법과 비슷하지만, 실제 JSON은 정해진 규칙을 지켜 작성해야 한다.

항목 설명
정의 텍스트 기반의 데이터 교환 형식이다. JavaScript 문법을 바탕으로 하지만 언어에 독립적으로 사용할 수 있다.
확장자 .json
구성 key와 value의 쌍으로 구성된다.
Key 특징 JSON 형식에서 key는 반드시 큰따옴표("")로 감싸야 한다.
Value 타입 문자열, 숫자, 불리언, 배열, 객체, null을 사용할 수 있다.
작성 방식 중괄호 {} 안에 내용을 작성하고, 각 쌍은 쉼표(,)로 구분한다. 
{
    "name": "홍길동",
    "age": 30,
    "isMember": true
}

3.7.2 JSON과 JavaScript 객체의 차이

JSON과 JavaScript 객체는 데이터를 key-value 형태로 작성한다는 점에서 비슷하지만, JSON은 데이터 교환을 위한 문자열 형식이고, JavaScript 객체는 프로그램 안에서 데이터를 다루기 위한 객체이다.

항목 JavaScript 객체 JSON
사용 목적 프로그램 안에서 데이터 구성 및 기능 처리 데이터 전송 또는 저장
Key 따옴표 선택 사항 필수이며 큰따옴표를 사용한다.
문자열 따옴표 작은따옴표와 큰따옴표 모두 사용 가능 큰따옴표만 사용한다.
함수 포함 여부 함수 포함 가능 함수 포함 불가 
- JavaScript 객체
const user = {
    name: '홍길동',
    age: 30,
    printInfo: function() {
        console.log(this.name);
    }
};

- JSON 문자열
const jsonData = '{"name":"홍길동", "age":30}';

3.7.3 기본 객체와 데이터 접근

JSON 형식의 데이터는 key-value 쌍으로 구성된다. 실제 개발에서는 JSON 문자열을 자바스크립트 객체로 변환하여 사용한다. 변환된 객체의 값은 점 표기법 또는 대괄호 표기법으로 접근할 수 있다. 

const 객체명 = {
    "key1" : "value1",
    "key2" : value2,
    "key3" : "value3"                
};

객체명.key1;
객체명["key1"];

객체명.key1 = "새로운 값";
객체명["key1"] = "새로운 값";

대괄호 표기법 사용 경우

키 이름에 공백이 있거나 변수에 저장된 키 이름으로 동적으로 접근할 때는 객체명["키"] 형태의 대괄호 표기법을 사용할 수 있다.

3.7.4 배열을 포함한 객체

객체의 값으로 배열을 저장할 수 있다. 배열에 접근할 때는 인덱스를 사용하며, 인덱스는 0부터 시작한다. 

const 객체명 = {
    "key1": "value1",
    "list": ["값1", "값2", "값3"]
};
            
객체명.list[0];
객체명.list[1];

      3.7.6 배열 안의 객체

      배열 안에는 여러 개의 객체를 저장할 수 있다. 게시글 목록, 회원 목록, 도서 목록처럼 같은 구조의 데이터를 여러 개 다룰 때 자주 사용한다.

      3.7.7 객체 리터럴과 메서드

      JavaScript 객체 리터럴은 JSON과 비슷한 문법으로 작성하지만, 함수도 포함할 수 있다는 차이가 있다. 객체 안에 정의된 함수를 메서드라고 하며, 객체의 데이터와 기능을 함께 묶어 사용할 수 있다.

    필드명: "값",

      3.7.8 JSON 문자열 변환

      JSON.parse()는 JSON 문자열을 JavaScript 객체로 변환하고, JSON.stringify()는 JavaScript 객체를 JSON 문자열로 변환한다.

      메서드 설명
      JSON.parse() JSON 문자열을 JavaScript 객체로 변환한다.
      JSON.stringify() JavaScript 객체를 JSON 문자열로 변환한다. 
      const jsonString = '{"name":"홍길동", "age":20}';
const userObject = JSON.parse(jsonString);

const userString = JSON.stringify(userObject);
