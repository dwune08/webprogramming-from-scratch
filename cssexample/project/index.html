HTML 기초

HTML은 웹 페이지의 구조를 만드는 언어이다. 시맨틱 태그를 사용하면 문서 구조를 의미 있게 표현할 수 있다.

오늘의 학습 기록

오늘은 HTML5 시맨틱 태그와 CSS 레이아웃을 함께 사용해 보았다. 시맨틱 태그로 문서 구조를 만들고, CSS의 flex와 grid로 화면 배치를 구성하였다.
