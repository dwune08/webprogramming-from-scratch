홈 HTML CSS JavaScript 학습 갤러리

학습 주제 갤러리

Grid Layout을 사용하여 여러 개의 항목을 일정한 간격으로 배치한 갤러리 예제입니다.

📄

HTML

HTML은 웹 페이지의 구조를 만드는 언어이며, 제목, 문단, 목록, 링크 등을 작성할 수 있습니다.

🎨

CSS

CSS는 웹 페이지의 디자인을 담당하며, 색상, 여백, 테두리, 배경 등을 설정할 수 있습니다.

⚙️

JavaScript

JavaScript는 웹 페이지에 동작을 추가하며, 버튼 클릭이나 화면 변경 기능을 만들 수 있습니다.

📦

박스 모델

박스 모델은 HTML 요소가 차지하는 영역을 content, padding, border, margin으로 설명합니다.

🧭

Flex Box

Flex Box는 요소를 한 방향으로 배치하고 정렬할 때 사용하는 CSS 레이아웃 방식입니다.

🔲

Grid Layout

Grid Layout은 웹 페이지를 행과 열로 나누어 요소를 배치하는 2차원 레이아웃 방식입니다.
