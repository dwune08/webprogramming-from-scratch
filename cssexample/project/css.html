홈 HTML CSS JavaScript 학습 갤러리

CSS의 정의와 역할

CSS(Cascading Style Sheets)는 HTML로 만든 웹 페이지의 뼈대에 옷을 입히고 꾸미는 스타일시트 언어입니다. 글꼴, 색상, 정렬, 배경뿐만 아니라 요소들의 배치와 레이아웃을 결정하여 사용자에게 아름다운 화면을 제공합니다.

CSS 선택자와 스타일 속성

CSS는 변경하고 싶은 HTML 요소를 콕 집는 '선택자(Selector)'와 무엇을 바꿀지 결정하는 '속성(Property)'으로 이루어집니다. id나 class를 이용해 특정 태그만 선택하고, 중괄호 { } 안에 스타일 규칙을 선언합니다.

박스 모델 (Box Model)

HTML의 모든 요소는 사각형의 상자(Box) 형태를 가집니다. 실제 콘텐츠 영역을 중심으로 안쪽 여백인 padding, 테두리인 border, 바깥쪽 여백인 margin으로 구성되며, 이들의 관계를 이해해야 레이아웃이 삐져나가는 현상을 막을 수 있습니다.

Flexbox 레이아웃 (1차원 정렬)

Flexbox는 행(가로) 또는 열(세로) 중 한 방향으로 요소를 유연하게 정렬할 때 사용하는 강력한 레이아웃 모델입니다. 메뉴 바를 나란히 배치하거나, 카드 내부의 아이템들을 정중앙에 정렬하고 정밀하게 간격을 분배할 때 핵심적으로 쓰입니다.

Grid 레이아웃 (2차원 배치)

Grid는 가로(행)와 세로(열)를 모두 사용하는 2차원 바둑판식 레이아웃을 만들 때 사용합니다. 페이지 전체의 메인 본문 영역과 사이드바 영역을 크게 나누거나, 정형화된 갤러리 형태의 화면 구조를 완벽하게 설계할 때 가장 효과적입니다.
