학습 과정 소개

웹 개발을 처음 학습할 때는 HTML로 문서의 구조를 만들고, CSS로 디자인을 적용한 뒤, JavaScript로 동적인 기능을 추가하는 흐름을 이해하는 것이 중요합니다.

아래 카드 목록은 웹 개발의 기본이 되는 세 가지 기술을 소개합니다. 각 기술의 역할과 주요 학습 내용을 확인해 보세요.

웹 개발 핵심 기술

HTML

HTML은 웹 페이지의 구조를 만드는 마크업 언어입니다. 제목, 문단, 목록, 이미지, 링크, 표와 같은 기본 요소를 작성할 때 사용합니다.

학습 단계: 기초
  • 기본 문서 구조 작성
  • 제목과 문단 태그 사용
  • 묵록, 이미지, 링크 작성
  • 시맨틱 태그 이해

CSS

CSS는 웹 페이지의 디자인을 담당하는 스타일 언어입니다. 글자 색상, 배경색, 여백, 테두리, 배치 등을 설정하여 화면을 보기 좋게 꾸밀 수 있습니다.

학습 단계: 기초 ~ 중급
  • 선택자 사용
  • 글자와 문단 스타일 적용
  • 박스 모델 이해
  • Flex와 Grid 레이아웃 학습

JavaScript

JavaScript는 웹 페이지에 동작을 추가하는 프로그래밍 언어입니다. 버튼 클릭, 입력값 검사, 화면 내용 변경 등 사용자와 상호작용하는 기능을 만들 수 있습니다.

학습 단계: 중급
  • 변수와 자료형 이해
  • 조건문과 반복문 사용
  • 함수 작성
  • DOM 요소 제어
