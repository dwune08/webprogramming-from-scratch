수강생 공지사항

이번 주 실습 과제는 HTML 시맨틱 태그와 CSS 레이아웃을 함께 사용하여 간단한 웹 페이지를 제작하는 것이다.

과제 제출 시에는 HTML 파일과 CSS 파일을 분리하여 제출해야 하며, 이미지 파일을 사용하는 경우 image 폴더 안에 저장한다.

파일명은 반드시 영문 소문자로 작성하고, 공백 대신 밑줄(_)을 사용한다. 예를 들어 main_page.html, common_style.css와 같이 작성한다.

제출 전에는 브라우저에서 화면이 정상적으로 출력되는지 확인하고, 링크 경로와 이미지 경로가 올바르게 연결되어 있는지 점검한다.

공지 내용이 박스 높이보다 많아질 수 있으므로 'overflow: auto;'를 사용하여 필요할 때만 스크롤바가 나타나도록 설정한다.
